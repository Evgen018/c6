# Калькулятор

Простой браузерный калькулятор с современным интерфейсом.

## Возможности

- Базовые математические операции (сложение, вычитание, умножение, деление)
- Поддержка десятичных чисел
- Очистка и удаление последнего символа
- Смена знака числа
- Поддержка клавиатуры

## Деплой на Vercel

Проект готов к деплою на Vercel. Для деплоя выполните следующие шаги:

1. Установите Vercel CLI (если еще не установлен):
```powershell
npm install -g vercel
```

2. Войдите в Vercel:
```powershell
vercel login
```

3. Задеплойте проект:
```powershell
vercel
```

Или подключите репозиторий к Vercel через веб-интерфейс:
- Перейдите на [vercel.com](https://vercel.com)
- Нажмите "Add New Project"
- Импортируйте ваш Git-репозиторий
- Vercel автоматически определит настройки и задеплоит проект

## Локальный запуск

Для локального запуска используйте любой статический сервер, например:

```powershell
npx serve .
```

или

```powershell
python -m http.server 8000
```

Затем откройте браузер по адресу `http://localhost:8000`
